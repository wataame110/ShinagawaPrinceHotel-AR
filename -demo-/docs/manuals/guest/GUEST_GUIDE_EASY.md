## これは なに？

スマートフォン（スマホ）で
**きれいな額縁（フレーム）つきの記念写真** が撮れる
無料のサービスです。

撮った写真は、スマホにそのまま保存できます。

---

## つかいかた

### ① QRコードを よみとる

テーブルにある **四角い白黒のもよう**（QRコード）を
スマホの **カメラ** でうつしてください。

画面の上に リンク（青い文字）が出てきます。
**そこをタップ**（指で軽くさわる）してください。

```
　QRコードが読めないときは？
　→ テーブルに書いてあるURL（インターネットの住所）を
　  直接入力してください。
　  わからなければスタッフにお声がけください。
```

### ② パスワードを いれる

画面が開いたら、パスワードを入力するところがあります。

**スタッフからお聞きしたパスワード** を入力して、
**「ログイン」ボタン** を押してください。

```
  パスワードが通らないときは？
  → パスワードは「半角」で入力してください
  → 大文字と小文字を間違えていないか確認してください
  → わからなければスタッフにお聞きください
```

### ③「カメラをゆるす」を おす

はじめて使うときだけ、
**「カメラへのアクセスを許可しますか？」** と聞かれます。

**「許可」** を押してください。

```
  まちがえて「許可しない」を押してしまったら？
  →  iPhone: 「設定」→「Safari」→「カメラ」→「許可」
  →  Android: ブラウザの「鍵のマーク」をタップ →「カメラ」→「許可」
  →  わからなければスタッフにお声がけください
```

### ④ しゃしんを とる

カメラの画面が出てきます。

画面の下にある大きな **「撮影する」ボタン** を押してください。

**3... 2... 1...** とカウントが出てから、
自動で写真が撮れます。

```
  こんなこともできます:

  ・フレーム（額縁）を変える
    → 画面の上にある四角いマークを押す

  ・写真の色を変える（セピアなど）
    → 画面の上にあるフィルターのマークを押す

  ・カメラを切り替える（自分撮り ←→ 向こう側）
    → 撮影ボタンの横にあるカメラのマークを押す

  ・メッセージを変える
    → 画面の上にある吹き出しのマークを押す

  ・言葉（言語）を変える
    → 画面の右上にある地球のマークを押す
    → 英語・中国語・韓国語 など9つの言葉に変えられます
```

### ⑤ しゃしんを ほぞん（保存）する

撮った写真が画面に表示されます。
**スマホに保存する方法が3つ** あります。

#### 【いちばん おすすめ】共有ボタンを使う

```
  1. 「共有」ボタン（□に↑のマーク）を 指で押す
  2. 出てきたメニューから「画像を保存」を 指で押す
  3. スマホの「写真アプリ」に保存されます！
```

#### 【おすすめ】写真を長押しする

```
  1. 画面に表示された写真を 指で長く押し続ける（3秒くらい）
  2. メニューが出てくる
  3. 「写真に保存」を 指で押す
  4. スマホの「写真アプリ」に保存されます！
```

#### 「保存する」ボタンを使う

```
  1. 「保存する」ボタンを 指で押す
  2. ダウンロードフォルダに保存されます
     （写真アプリではなく、ファイルアプリの中です）
```

### ⑥ もういちまい とる

**「再撮影」ボタン** を押すと、
もう一度写真が撮れます。

**何回でも撮れます。** 回数に制限はありません。
