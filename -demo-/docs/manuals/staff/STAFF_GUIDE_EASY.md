## この撮影サービスって なに？

お誕生日や記念日に レストランに来てくださったお客様が、
スマートフォンで **記念写真** を撮れるサービスです。

写真には、レストランごとのオリジナルの
**かわいいフレーム（額縁）** が自動でつきます。

```
お客様がやること:

  ① QRコードを読む
  ② パスワードを入れる
  ③ 写真を撮る
  ④ スマホに保存する

これだけです！
```

## こまったとき どうする？

### お客様から「QRコードが読めない」と言われたら

1. **カメラアプリ** で読み取っているか確認
   - iPhone → 標準の「カメラ」アプリを使う
   - Android → 標準の「カメラ」アプリを使う

2. スマホとQRコードの **距離** を調整する
   - 近すぎても遠すぎてもダメです
   - だいたい **15cm〜30cm** くらいがちょうどいいです

3. それでもダメなら → URLを直接入力してもらう
   （QRコードの台紙にURLが書いてあります）

### 「カメラが映らない」と言われたら

1. 「カメラを許可しますか？」という画面が出たら **「許可」** を押してもらう

2. もし **「許可しない」** を押してしまった場合:
   - iPhone → 「設定」→「Safari」→「カメラ」→「許可」
   - Android → ブラウザの **鍵のマーク** を押す →「カメラ」→「許可」

3. LINEや電話など **他のアプリがカメラを使っていないか** 確認

### 「写真が保存できない」と言われたら

```
【iPhone のお客様】
  → 「共有」ボタン（□に↑のマーク）を押す
  → 「画像を保存」を押す

【Android のお客様】
  → 「保存する」ボタンを押す
  → ダウンロードフォルダに保存される

【どちらでも使える方法】
  → 撮った写真を指で長く押し続ける
  → 「写真に保存」を押す
```

### 画面が変・おかしい場合

1. ブラウザ（SafariやChrome）を **完全に閉じる**
2. もう一度 QRコードを読み取る
