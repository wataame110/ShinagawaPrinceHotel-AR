## 2. サービス概要

記念日（お誕生日・記念日等）にご来店されたお客様に、
レストランオリジナルのフォトフレーム付き写真を撮影いただけるサービスです。

- お客様のスマートフォンのみで完結します
- アプリのインストールは不要です
- 撮影した写真はお客様の端末に直接保存されます
- データの通信料はお客様負担（Wi-Fi環境推奨）

## 4. お客様ご案内の流れ

### ステップ1: QRコード読み取り

お客様にテーブル上のQRコードをスマートフォンのカメラで読み取っていただきます。

**iPhone の場合:** 標準のカメラアプリでQRコードを映すと、上部に通知が表示されます。
**Android の場合:** カメラアプリまたはGoogleレンズでQRコードを読み取れます。

### ステップ2: パスワード入力

ログイン画面が表示されたら、パスワードをお客様にお伝えください。
お客様がパスワードを入力し「ログイン」をタップすると、撮影画面に進みます。

### ステップ3: カメラの許可

初回アクセス時に「カメラへのアクセスを許可しますか？」と表示されます。
**「許可」を選んでいただくようお声がけください。**

### ステップ4: 撮影

撮影画面でお客様ご自身で操作していただきます。以下の機能が使えます:

| 機能 | 操作方法 |
|------|---------|
| 撮影 | 画面下部の「撮影する」ボタンをタップ → 3秒カウントダウン → 自動撮影 |
| フレーム選択 | 画面上部のフレームボタン → お好きなフレームを選択 |
| フィルター | 画面上部のフィルターボタン → 色味の変更（セピア、モノクロ等） |
| Face AR | 画面上部のARボタン → 顔にデコレーション |
| メッセージ編集 | 画面上部の吹き出しボタン → 日付やメッセージの変更 |
| カメラ切替 | 撮影ボタン横のカメラアイコン → インカメ/アウトカメ切替 |
| 言語切替 | 画面右上の地球儀アイコン → 9言語対応 |

### ステップ5: 写真の保存

撮影後、お客様に保存方法をご案内してください:

| 方法 | おすすめ度 | 手順 |
|------|----------|------|
| **共有ボタン（推奨）** | ★★★ | 「共有」ボタンをタップ → 「画像を保存」を選択 → 写真アプリに保存 |
| **長押し保存** | ★★★ | 撮影した写真を指で長押し → 「写真に保存」を選択 |
| **保存ボタン** | ★★☆ | 「保存する」ボタンをタップ → ダウンロードフォルダに保存 |

**一番おすすめ:** iPhoneのお客様には「共有ボタン」または「写真の長押し」をご案内ください。

## 6. よくある質問（FAQ）

### Q. お客様がQRコードを読み取れない
**A.** 以下をお試しください:
1. カメラアプリ（標準カメラ）で読み取っているか確認
2. 画面の明るさを上げてもらう
3. QRコードとスマホの距離を調整してもらう
4. それでもダメならURLを直接入力（URLはQRコードの台紙に記載）

### Q. 「カメラへのアクセスを許可」で「許可しない」を押してしまった
**A.** 端末の設定から変更できます:
- **iPhone:** 設定 → Safari → カメラ → 「許可」に変更
- **Android:** Chrome → アドレスバー横の鍵マーク → カメラ → 「許可」

### Q. 撮った写真がどこに保存されたかわからない
**A.** 保存方法によって保存先が異なります:
- 「共有」→「画像を保存」: 写真アプリ
- 「保存する」ボタン: ダウンロードフォルダ（iPhoneは「ファイル」アプリ内）
- 長押し →「写真に保存」: 写真アプリ

### Q. 画面が英語（他の言語）になっている
**A.** 右上の地球儀アイコンをタップし、「日本語」を選択してください。

### Q. フレームが表示されない
**A.** Wi-Fi接続を確認してください。初回は画像の読み込みに数秒かかります。
改善しない場合はブラウザを一度閉じて再度QRコードを読み取ってください。

### Q. 撮影回数に制限はありますか？
**A.** ありません。何度でも撮影できます。

### Q. このサービスにお客様の費用はかかりますか？
**A.** 無料です。ただし通信料はお客様のご負担となります（Wi-Fi環境推奨）。

## 7. トラブル発生時の対応

### まずやること

1. **ブラウザを閉じて、もう一度QRコードを読み取ってもらう**
2. **Wi-Fi接続を確認する**
3. **端末を再起動してもらう**

### それでも解決しない場合

| 症状 | 対処 |
|------|------|
| 画面が真っ白 | サーバー側の問題の可能性。管理者に連絡 |
| ログインできない | パスワードの大文字/小文字・全角/半角を確認 |
| カメラが映らない | 他のアプリ（LINE通話等）がカメラを使っていないか確認 |
