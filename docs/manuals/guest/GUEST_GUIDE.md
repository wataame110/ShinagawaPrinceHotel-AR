## ご利用の流れ

### STEP 1. QRコードを読み取る

テーブルに置かれたQRコードを、スマートフォンのカメラで読み取ります。

- **iPhone:** 標準のカメラアプリでQRコードを映すと、画面上部にリンクが表示されます。タップして開いてください。
- **Android:** カメラアプリでQRコードを映すと、リンクが表示されます。タップして開いてください。

### STEP 2. パスワードを入力する

ログイン画面が表示されます。
スタッフからお伝えするパスワードを入力し、「ログイン」をタップしてください。

### STEP 3. カメラのアクセスを許可する

初回のみ「カメラへのアクセスを許可しますか？」という画面が表示されます。
**「許可」を選択** してください。

### STEP 4. 撮影する

カメラ画面が表示されたら、撮影の準備完了です。

| 操作 | 方法 |
|------|------|
| **撮影** | 画面下部の「撮影する」ボタンをタップ → 3秒カウントダウン後に自動撮影 |
| **フレーム選択** | 画面上部の四角いアイコンをタップ → お好きなデザインを選択 |
| **フィルター** | 画面上部のフィルターアイコンをタップ → セピア・モノクロなど10種類 |
| **Face AR** | 画面上部のARアイコンをタップ → 顔にデコレーションを追加 |
| **メッセージ編集** | 画面上部の吹き出しアイコンをタップ → 日付・メッセージ・場所を編集 |
| **カメラ切替** | 撮影ボタン横のカメラアイコンをタップ → インカメラ/アウトカメラ切替 |
| **言語切替** | 画面右上の地球儀アイコンをタップ → 9言語対応 |

### STEP 5. 写真を保存する

撮影後のプレビュー画面で、以下の方法で保存できます。

**方法A: 共有ボタン（おすすめ）**
1. 「共有」ボタン（📤マーク）をタップ
2. 「画像を保存」を選択
3. 写真アプリに保存されます

**方法B: 写真を長押し（おすすめ）**
1. プレビュー写真を指で長押し
2. 「写真に保存」を選択
3. 写真アプリに保存されます

**方法C: 保存ボタン**
1. 「保存する」ボタンをタップ
2. ダウンロードフォルダに保存されます

### STEP 6. もう一枚撮る

「再撮影」ボタンをタップすると、もう一度撮影できます。
撮影回数に制限はありません。何度でも撮れます。

## よくあるご質問

### Q. 撮影にお金はかかりますか？
**A.** 無料でご利用いただけます。ただし、通信にかかるデータ通信料はお客様のご負担となります。

### Q. 対応していないスマートフォンはありますか？
**A.** iPhone（iOS 14以降）およびAndroid（Android 8以降）に対応しています。
一部の古い端末では正常に動作しない場合があります。

### Q. 撮影した写真はホテルに保存されますか？
**A.** いいえ。お客様の端末にのみ保存されます。ホテル側にデータは残りません。

### Q. フレームなしで撮影できますか？
**A.** はい。フレーム選択画面で「フレームなし」を選んでいただけます。

### Q. カメラが起動しません
**A.** 以下をお試しください:
1. 「カメラへのアクセスを許可」の画面で「許可」を選んでいるか確認
2. 他のアプリ（LINE通話など）がカメラを使用中でないか確認
3. ブラウザを一度閉じて、再度QRコードを読み取る
