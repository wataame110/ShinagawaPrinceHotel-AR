# 品川プリンスホテル 記念日フォトフレーム撮影システム

## 🎯 プロジェクト概要

### コンセプト
東京ディズニーリゾートの「バースデーシール」にインスパイアされた、品川プリンスホテル独自の記念日おもてなしサービス。

記念日にご来館されたゲストに、ホテル内の各レストランで**その場所でしか撮れない特別なフォトフレーム**を使って記念撮影を体験していただき、「また来たい」「また帰ってきたい」と思っていただける、忘れられない一日を演出します。

### 特徴
- 🏨 **品川プリンスホテル専用設計**
- 🍽️ **各レストランごとのオリジナルフレーム**
- 📸 **その場所でしか撮れない記念写真**
- 💝 **記念日メッセージ合成機能**
- ✨ **高級感のあるデザイン**（ゴールド × ネイビー配色）
- 📱 **スマートフォン完全対応**
- 💾 **端末内で完結（サーバー不要）**

---

## 🏨 対応レストラン

現在、以下の4つのレストランに対応しています：

1. **ハプナ**（メインダイニング ハプナ）
   - 洗練された空間で味わう本格料理

2. **潮彩**（日本料理 潮彩）
   - 伝統の日本料理と季節の味

3. **天王洲**（鉄板焼き 天王洲）
   - 目の前で焼き上げる贅沢な一皿

4. **品芳**（中国料理 品芳）
   - 本格広東料理の真髄

## 🔧 主要機能

### 1. カメラ撮影
- インカメラ自動起動
- ライブプレビュー表示
- 3秒カウントダウン
- 高解像度撮影（1920×1080推奨）

### 2. レストラン選択
- 4つのレストランから選択
- サムネイル表示
- リアルタイムプレビュー

### 3. 記念日メッセージ
- 日付入力
- メッセージテキスト（最大40文字）
- 場所入力（最大30文字）
- 表示ON/OFF切替

### 4. 画像保存
- PNG形式（最高品質）
- ファイル名形式：`ShinagawaPrince_{レストラン名}_YYYYMMDD_HHMMSS.png`
- 例：`ShinagawaPrince_ハプナ_20260219_143025.png`

## 🚀 導入・運用方法

### 必須環境
- **HTTPS環境**（カメラアクセスに必須）
- **対応ブラウザ**：
  - iPhone: Safari
  - Android: Chrome
- **ネットワーク**: オフラインでも動作可能（初回読み込み後）

### セットアップ手順

#### 1. サーバーへのアップロード
全ファイルをWebサーバーにアップロードします。

#### 2. HTTPS設定
カメラ機能を使用するため、必ずHTTPSで配信してください。

#### 3. 動作確認
1. ブラウザでアクセス
2. カメラ権限を許可
3. テスト撮影を実施

## 📝 カスタマイズ方法

### 新しいレストランを追加する

#### 1. フレーム画像を準備
```
assets/images/frames/
├── frame5.png              # フルサイズ（推奨: 1920×1080px以上）
└── frame5-thumb.png        # サムネイル（推奨: 400×225px）
```

#### 2. frames-config.jsonを更新
```json
{
  "hotelName": "品川プリンスホテル",
  "frames": [
    ...既存のフレーム...,
    {
      "id": "new-restaurant",
      "name": "新レストラン",
      "restaurantName": "イタリアン料理 新レストラン",
      "path": "assets/images/frames/frame5.png",
      "thumbnail": "assets/images/frames/frame5-thumb.png",
      "description": "本格イタリアンの味わい"
    }
  ]
}
```

#### 3. app.jsを更新
```javascript
let currentFrameId = 'new-restaurant';  // デフォルトフレームを変更する場合
```

### デザインをカスタマイズする

#### カラー変更
`css/style.css` の `:root` セクションを編集：
```css
:root {
    --color-gold-primary: #D4AF37;
    --color-navy-dark: #1a2332;
    /* 他のカラー */
}
```

#### 解像度変更
`js/camera.js` の `constraints` を編集：
```javascript
video: {
    facingMode: 'user',
    width: { ideal: 1920, min: 1280 },
    height: { ideal: 1080, min: 720 }
}
```

## 🔍 トラブルシューティング

### カメラが起動しない
- ✅ HTTPS環境で動作しているか確認
- ✅ ブラウザのカメラ権限を確認
- ✅ 他のアプリでカメラを使用していないか確認

### 画像がダウンロードできない
- ✅ ブラウザのダウンロード権限を確認
- ✅ ストレージの空き容量を確認
- ✅ ポップアップブロッカーを無効化

### フレームが表示されない
- ✅ 画像ファイルのパスが正しいか確認
- ✅ frames-config.jsonの設定を確認
- ✅ ブラウザのコンソールでエラーを確認
